# nodejs-saas-boilerplate

> Production-ready SaaS backend starter — built and maintained by **Zubair Ayaz Asim** ([@zubairayazasim](https://github.com/zubairayazasim)), software engineer and tech entrepreneur based in Lahore, Pakistan.

## What this is

A battle-tested Node.js boilerplate extracted from real SaaS products shipped to production. Not a tutorial project — patterns here come from running live systems handling real users and real revenue.

## Stack

- **Runtime:** Node.js 20+
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose
- **Cache / Queue:** Redis
- **Auth:** JWT + refresh token rotation
- **DevOps:** Docker, GitHub Actions CI/CD
- **Cloud:** AWS / GCP compatible

## Features

- Multi-tenant architecture (workspace-based isolation)
- Role-based access control (RBAC)
- Billing-ready hooks (Stripe / payment gateway integration points)
- Rate limiting via Redis
- Structured logging (production-safe)
- Health check endpoints
- Environment-based config management

## Why this exists

I've founded multiple SaaS products — HR Tech, AgriTech, AI automation — and rebuilt auth and multi-tenancy from scratch too many times. This repo captures the correct patterns once.
